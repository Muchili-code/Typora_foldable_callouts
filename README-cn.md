# Typora 可折叠警示框
> [!note]
> 这是我为 Typora 添加的个人标注功能，支持 Typora 中的标注折叠，类似于 Obsidian 的功能。

## 问答
**我的 Typora 版本是否支持？**
Typora 版本应 ≥ 0.9.98（最后一个免费版本）。

## 使用平台
由于技术原因，目前仅在 Windows 平台上开发。

## 使用方法
首先，根据教程下载相应的插件。找到下载插件所在的 `./resources/plugin/custom/plugins` 目录，并将 `callout.js` 的全部内容替换为 [发布版本](https://github.com/Muchili-code/Typora_foldable_callouts/releases) 中的 `callouts.js` 内容。

然后，启用插件的标注功能支持（下载后默认启用）。

可选：禁用 Typora 的默认标注支持。这会避免Typora默认的css样式与插件的样式之间的冲突

## 支持的语法
`> [!NOTE]+ "描述"`, 默认情况下视为折叠状态。
